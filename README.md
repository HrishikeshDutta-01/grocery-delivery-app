A full-stack grocery delivery web application with product listing, cart functionality, secure user authentication, and order placement features (payment integration pending). The project follows a clean client + server architecture built using modern web technologies.
 # Features
User Features

Browse grocery products

Add/remove items from cart

View total cart price

User signup & login (JWT authentication)

Place an order (without payment gateway)

Admin Features

Add new products

Edit or delete products

Manage orders

View customer details

# Tech Stack
Frontend (Client)

HTML, CSS, JavaScript

React / Vite (if used)

Responsive UI

Backend (Server)

Node.js

Express.js

JWT Authentication

MongoDB (Mongoose ORM)
